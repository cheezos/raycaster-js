# Javascript Raycaster

A simple raycaster using Javascript, PixiJS and Electron.

### Setup

Clone the repository and open it in VSCode/Terminal.

Run these commands in order:

1. `npm i -D electron`
2. `npm i pixi.js`
3. `npm run start`
