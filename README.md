## Javascript Raycaster

Doom-style raycaster in Javascript using PixiJS and Electron.

## Setup & Run

1. Clone and open the project in a terminal.
2. Install the dependencies `npm install`.
3. Run `npm run start`.

## Controls

- Move with `W` `A` `S` `D`.
- Mouse to look around.
